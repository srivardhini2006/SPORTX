# SPORTX

A production-ready sports e-commerce platform built to learn and demonstrate modern software engineering practices.

## 🎯 Project Goal

SPORTX is more than an e-commerce application. It is a full-stack software engineering project designed to gain hands-on experience in:

- React
- Spring Boot
- PostgreSQL
- Docker
- Kubernetes
- CI/CD
- REST APIs
- Software Architecture
- System Design

The focus is on building a production-quality application while understanding the purpose behind every technology used.
